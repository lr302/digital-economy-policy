# 中国数字经济与贸易政策知识库

## 概述

本知识库系统梳理了 **2015-2026年** 中国数字经济与贸易领域的 **50条核心政策**，
涵盖七大方向，分为四个演变时期。

## 研究范围

- **时间范围**：2015年（互联网+行动）至2026年（十五五开局）
- **政策类型**：法律、行政法规、部门规章、地方性法规
- **法域**：中国

## 典型政策路径

以 **数据治理** 方向为例的政策演进：

```
2015 大数据行动纲要 → 2016 网络安全法
    ↓
2021 数据安全法 + 个人信息保护法
    ↓
2022 数据出境安全评估 + 数据二十条
    ↓
2023-24 个人信息出境标准合同 + 跨境流动规定
    ↓
2025 数据基础设施建设指引
```

## 使用方式

1. **阅读分析报告**：`数字经济政策演变趋势分析报告.md`
2. **浏览政策笔记**：在 `知识笔记/` 中按编号或时期查看
3. **Obsidian 关联**：将此文件夹作为 Obsidian Vault 打开，利用图谱和反向链接功能
4. **教学使用**：配合课程内容，选取对应政策的笔记作为阅读材料
